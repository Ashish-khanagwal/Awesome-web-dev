Code For Community

Founded by: Ashish khanagwal

Code For Community's agenda is to help folks to Kickstart their Open Source journey. Open Source is for everyone, if you're willing to start your Open Source journey then tech-stack, work experience doesn't matter. Contributing to Open Source can be overwhleming at the beginning, you just have to find some good first issues to get started and for some hands on practice. You can achieve that by fixing typos, contributing to no-code repos, maintaining the Documentaions.
